## 字符串处理和print

- 字符粗是一个 seq，所以可以用 ch in str、[]

倒序输出。str[::-1]

- 引号：单引号，双引号，三引号。

- r + 字符串。原样输出，不会发生转义。 print(r'\n') 

- 占位符1。'age=%d,name=%s,pi=%5.2f'%(12,'mdk',3.14)

## 语句

- 条件语句

```python
if bool1:
    exp1
elif bool2:
    exp2
else:
    exp3
```

- for 循环语句

```python
for var in seq:
    exp
```

```python
# [0,9] 循环
for i in range(0,10):
    print(i,end=' ')
```

```python

    exp1
else: # 循环正常结束执行，不正常结束(break)就不执行
    exp2
```

- while 循环语句

```python
while bool:
    exp
```

- pass 空语句。相当于 ;

## 列表 list

- 空列表。arr = []

- 构建列表。arr=[1, 2]

- arr[-1] 最后一个元素

- len(arr)。即 arr.length

- ele in arr。即 arr.contains(ele)

- del arr[1]。删除第一个元素，相当于arr.remove(ele)，或者 arr.pop(1)

- 切片。就是 subList

- 尾加元素。arr.append(ele) 

- 插入。arr.insert(i, ele)

- 添加数组。arr.extend(arr2)，或者 arr + arr2

- arr.sort() 排序。

## 元组 tuple

- 空元组。arr=()

- 列表转元组。tuple(list)

- 拆包。a,b = (1, 2)

- 不定拆包。a,*b,c = (1,2,3,4,5,6)。a=1, c=6，b=[2,3,4,5]

- 不定拆包2。a,*b = (1,)，则 a=1, b=[]
